|----------|----------|-----------------|-------------|--------------  |

| Transaction | paymentMethod | Спосіб оплати | Рядок | Немає         |

| Budget     | categories | Категорії бюджету | Список   | Немає         |

| Statistics | income | Дохід              | Дійсне число | Немає         |
